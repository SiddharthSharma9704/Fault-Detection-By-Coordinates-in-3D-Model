ðŸ“„ Project Title: Fault Detection by Coordinates in 3D Model

This project aims to automatically detect faults in 3D models by analyzing vibration sensor data and coordinate data from .stl files using machine learning.

1. Introduction

The main objective is to classify whether a fault is present at given coordinate points of a 3D object using supervised ML models trained on data collected from an ADXL345 sensor.

2. Dataset

  • X-direction: X-axis vibration/coordinate value
  • Y-direction: Y-axis vibration/coordinate value
  • Z-direction: Z-axis vibration/coordinate value
  • Error_found: Indicates if a fault was found (0 = No, 1 = Yes)

3. Libraries & Technologies

  • Python: pandas, numpy, matplotlib, seaborn, scikit-learn, xgboost, imblearn, numpy-stl, pickle
  • Deployment: Flask, HTML templates
  • Tools: Jupyter Notebook / Python IDE

4. Data Preprocessing

  • Handled missing values & outliers using median replacement
  • Visualized distributions (KDE, scatter plots)
  • Balanced data using ADASYN

5. Exploratory Data Analysis

  • KDE plots, scatter plots, violin plots
  • Heatmaps for correlation
  • Boxplots to detect/treat outliers

6. Model Training

Tested models: Logistic Regression, Random Forest, XGBoost, Naive Bayes, KNN, SVC

Hyperparameter tuning via GridSearchCV

7. Model Evaluation

  • Classification report, accuracy, F1 score, ROC AUC
  • Compared models using bar plots
  • Saved best model as model.pkl

8. Backend Deployment

  • Flask routes for homepage, predictor, prediction result
  • STL upload & coordinate extraction
  • Model prediction and result display

9. STL File Handling

  • Files saved temporarily with UUID
  • Coordinates extracted using numpy-stl
  • Predictions shown for first 1000 points
  • File deleted after use

10. Results

Each point in the STL file is predicted as "Fault Detected" or "No Fault". Results shown in table format for quick feedback.

11. Future Improvements

  • Expand dataset with more real-world readings
  • 3D visualization of faulty regions
  • User login/auth system
  • Cloud deployment for scalability
